Functional Programming

We are focussed on pushing OCaml and functional programming forward as a platform, making it a more effective tool for all users.

We work on all aspects of the OCaml developer toolchain, specifically:

  • The OCaml compiler: language, runtime and multicore support.
  • The OCaml Platform: a coherent set of tools and documentation.
  • MirageOS: a library OS for building secure, high-performance network applications.
